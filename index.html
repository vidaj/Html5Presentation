Hvordan komme i gang med HTML5

<!DOCTYPE html>
<html lang="no">
    <head>
        <meta charset="UTF-8">
    </head>
    <body>
    </body>
</html>

Som er bedre enn tidligere:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
                "http://www.w3.org/TR/html4/strict.dtd">
<html lang="no">
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    </head>
    <body>
    </body>
</html>

Canvas



    var canvas = document.getElementById('myCanvas');
    var image = document.getElementById('myImage');

    var context = canvas.getContext('2d');

    context.save();

    context.scale(2.0, 2.0);

    context.drawImage(image, 0, 0);

    context.restore();

Canvas - drawing


    var canvas = document.getElementById('myCanvas');
    var context = canvas.getContext('2d');

    context.save();

    context.fillStyle = "rgb(200, 0, 0)";
    context.fillRect(10, 10, 55, 50);

    context.restore();

Forms

Autofocus

Det er må mulig å spesifisere i markup hvilket element som automatisk får fokus. 

    

        <input name="q" autofocus>
        <input type="submit" value="Search">

Fungerer input (bortsett fra når type = hidden), select, textarea og button.

Placeholder

Native støtte for placeholder i input-felter

    <input type="email" placeholder="your.email@example.com>

Form-attributtet

Tidligere: 

    <form id="myForm">
        <label for="email">Email:</label>
        <input type="email" name="email">
    </form>

Nå er dette mulig:

    <label>
        <input type="email" form="myForm" name="email">
    </label>
    <form id="myForm></form>

Required-attributtet

<form id="form_with_required" action="#" >
    <label>Email:
        <input type="email" name="email" required>
    </label>
    <button type="submit">Submit</button>
</form>

GeoLocation



    navigator.geolocation.getCurrentPosition(callback);

    function callback(position) {
        var latitude = position.coords.latitude;
        var longitude = position.coords.longitude;
        var accuracy = position.coords.accuracy;

        position.coords.altitude;
        position.coords.altitudeAccuracy;
        position.coords.heading;
        position.coords.speed:
        position.timestamp;
    }

GeoLocation



    navigator.geolocation.getCurrentPosition(callback, handle_error);

    function handle_error(err) {
        switch (err.code) {
            case 1: //PERMISSION_DENIED
            case 2: //POSITION_UNAVAILABLE
            case 3: //TIMEOUT
            case 0: //UNKNOWN_ERROR
        }
    }
